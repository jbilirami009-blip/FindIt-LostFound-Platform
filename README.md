# FindIt - Lost & Found Platform

## 🚀 Overview
FindIt is a full-stack Lost & Found web application that allows users to report, search, and manage lost or found items.

The platform supports user interaction, image uploads, and administrative moderation.

---

## 🛠️ Tech Stack
- PHP
- MySQL
- HTML / CSS
- JavaScript

---

## ✨ Features
- Create, update, and delete posts
- Upload images for items
- Search and filter items
- User communication system
- Admin dashboard for moderation

---
